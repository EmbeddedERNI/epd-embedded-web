ESP32 Project Zero
Setup the ESP32 Development Environment

To install the toolchain and the Espressif IoT Development framework on Linux, Window or Mac Os, follow the official documentation on http://esp-idf.readthedocs.io/en/latest/get-started/index.html.

Install the toolchain

  1. Create a directory named esp at your home
  2. Download the xtensa-esp32 toolchain for raspberry
  3. Extract it under your ~/esp/xtensa-esp32-elf/ directory
  4. Check if ~/esp/xtensa-esp32-elf/bin is in your PATH, and if not, include it.
    For that, edit your ~/.profile or ~/.bash_profile file, and add the following line:
    export PATH="$PATH:$HOME/esp/xtensa-esp32-elf/bin"

Install your Epressif IoT Development Framework

  1. Download the Espressif IoT Development Framework
  2. Extract it under your ~/esp/esp-idf/ directory
  3. Add an environment variable IDF_PATH to your ~/.profile or ~/.bash_profile file.
    export IDF_PATH=~/esp/esp-idf
  4. Add your user to the dialout user group
    sudo usermod -a -G dialout $USER

Check USB Comm

  1. Run the command ls /dev/tty* before plugging in the USB cable
  2. Plug the USB cable to a free USB port in the raspberry and to the Espressif Board
  3. Run again the command ls /dev/tty* after plugging in the USB cable
  4. Write down the new entry. It should be /dev/ttyUSBx.
  5. Run a serial terminal to this port, with baudrate of 115200 and 8N1
    i.e.: miniterm.py /dev/ttyUSB0 115200
Reset your board. You should see the board log. If so, congrats!
