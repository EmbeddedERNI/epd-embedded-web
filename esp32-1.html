ESP32 Project One
Hello World

Get the code

  1. Copy ~/esp/esp-idf/examples/get-started/hello_world/ to ~/esp/
  2. Change directory to ~/esp/esp-idf/hello_world/

Hands-on

These steps are common for almost all projects.
  1. Run make menuconfig to configure the project
    1. Verify the communication port under "Serial flasher config -> Default serial port" menu entry, and change it if necessary
    2. Save and Exit
  2. Generate the binary using make
  3. Upload the binary using make flash
  4. Connect to the ESP log stream using make monitor. To exit the monitor, press Ctl + ]
You should see a Hello world! message and a 10 seconds countdown. If so, congrats!
